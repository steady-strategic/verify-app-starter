# Tailwind Classes Reference for Blog Feed

## Container Classes
- `.blog-container` - Main vertical flex container with 64px gap
- `.blog-row` - Horizontal flex row with 80px gap, center-aligned items
- `.blog-content` - Vertical flex column for text content with 24px gap

## Component Classes
- `.blog-mockup` - Dark primary background box with border radius and shadow
- `.blog-title` - 28px bold text with primary dark color
- `.blog-body` - 18px regular text with secondary gray color
- `.blog-button-group` - Flex row container for button with border and padding
- `.blog-button-icon` - SVG icon styling with dark secondary background

## Utility Classes Used

### Colors
- `text-text-primary` - #111928
- `text-text-secondary` - #6b7280
- `border-border` - #e5e7eb
- `bg-dark-primary` - #111928
- `fill-dark-secondary` - #1f2a37

### Spacing
- `gap-container-gap` - 64px
- `gap-row-gap` - 80px
- `gap-column-gap` - 24px
- `gap-button-gap` - 8px

### Border & Shadow
- `rounded-blog-lg` - 8px border radius
- `shadow-blog-card` - Card shadow effect

## Typography Classes
- `text-blog-title` - 28px, weight 700, letter spacing -0.56px, line height 35px
- `text-blog-body` - 18px, weight 400, line height 27px
- `text-blog-button` - 14px, weight 500

## Responsive Breakpoints
Currently no specific responsive breakpoints are defined. The layout uses flexbox which naturally adapts.

## Custom CSS Classes
The following custom classes are defined in `tailwind.css`:
- `.blog-container`
- `.blog-row`
- `.blog-mockup`
- `.blog-content`
- `.blog-button-group`
- `.blog-button-icon`
